This is a web clone of the Formula 1 Website that I work on in my free time as a fun little side project.

Skills

Project Purpose

The whole point of cloning this website is because I want to improve my Frontend skills and see if I can replicate existing designs as closely as possible.

Problems and Solutions

Because Formula 1 have copyrighted their fonts and images, it was difficult to get them and some I was unable to use so it cannot be a 100% accurate web clone but it is as close as I can make it.
